## 🛒 SHOPNOW — Detailed Project Description

**SHOPNOW** is a full-stack e-commerce web application built with Django that supports **multi-role interaction (Customer, Seller, Admin)**. The system enables product discovery, ordering, seller-side management, and admin moderation within a structured workflow.

---

# 🎯 System Overview

* **Architecture:** Django MVC (Templates + Views + Models)
* **Roles:** Customer, Seller, Admin
* **Core Idea:** Controlled marketplace with **approval-based product publishing**
